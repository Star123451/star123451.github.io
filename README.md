# peg86
peg86 uses [v86](https://github.com/copy/v86) to emulate an x86 CPU running ReactOS to play Peggle Deluxe/Nights in your browser.

[![Play Peggle Deluxe](https://img.shields.io/badge/Play%20Peggle%20Deluxe-orange)](https://sinceohsix.github.io/peg86) [![Play Peggle Nights](https://img.shields.io/badge/Play%20Peggle%20Nights-purple)](https://sinceohsix.github.io/peg86?nights)

## Compatibility
Compatibility is widely untested. If you have a decent computer, it *should* work. Mobile is not properly supported (yet). A list of tested devices can be found below.

| Device        | Browser | OS Version      | Runs? |
| :-----------: | :-----: | :-------------: | :---: |
| M2 Mac Mini   | Safari  | 26.3.1          | Great |
| Macbook Neo   | Safari  | 26.3.2          | Great |
| iPad 10th Gen | Safari  | 26.1            | Good  |
| Surface Pro 4 | Edge    | Windows 10 22H2 | Bad.  |
